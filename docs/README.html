🎓 MindSchool - Sistema de Gestión de Aprendizaje

MindSchool es un sistema de gestión de aprendizaje (LMS) moderno y completo, desarrollado con las mejores tecnologías web. Permite la administración de cursos, gestión de estudiantes y profesores, y el seguimiento del progreso académico.

🚀 Características Principales

👨‍🏫 Panel de Profesores

  • Dashboard moderno con mascota animada
  • Gestión de cursos y materias
  • Navegación por cuatrimestres
  • Visualización de grupos y estudiantes
  • Interfaz intuitiva y responsiva

👨‍🎓 Panel de Estudiantes

  • Dashboard con tareas pendientes
  • Lista de materias en formato Netflix
  • Tabla visual de tareas
  • Filtros por materia y estado
  • Diseño moderno y funcional

👨‍💼 Panel de Administrador

  • Gestión completa de usuarios
  • Administración de cursos
  • Reportes y estadísticas
  • Control de roles y permisos

🛠️ Tecnologías Utilizadas

Backend

🚀 Instalación y Configuración

Requisitos Previos

  • PHP 8.2+
  • Composer
  • Node.js 18+
  • MySQL 8.0+
  • MongoDB 6.0+

1. Clonar el Repositorio 

git clone https://github.com/tu-usuario/mindschool.git
cd mindschool

2. Configurar Backend (Laravel) 

cd backend
composer install
cp .env.example .env
php artisan key:generate
php artisan migrate
php artisan db:seed
php artisan serve

3. Configurar Frontend (React) 

cd frontend
npm install
npm run dev

4. Configurar Base de Datos 

# MySQL
mysql -u root -p
CREATE DATABASE mindschool;
GRANT ALL PRIVILEGES ON mindschool.* TO 'mindschool_user'@'localhost';

# MongoDB
mongo
use mindschool

🔄 API Endpoints

Autenticación

POST /api/register     - Registro de usuario
POST /api/login        - Inicio de sesión
POST /api/logout       - Cerrar sesión
GET  /api/user         - Obtener usuario actual

Cursos

GET    /api/cursos              - Listar cursos
POST   /api/cursos              - Crear curso
GET    /api/cursos/{id}         - Ver curso específico
PUT    /api/cursos/{id}         - Actualizar curso
DELETE /api/cursos/{id}         - Eliminar curso

Profesor

GET /api/profesor/dashboard           - Dashboard del profesor
GET /api/profesor/cursos              - Cursos del profesor
GET /api/profesor/cursos/{id}/estudiantes - Estudiantes del curso

Estudiante

GET /api/estudiante/dashboard      - Dashboard del estudiante
GET /api/estudiante/materias       - Materias del estudiante
GET /api/estudiante/tareas-pendientes - Tareas pendientes
